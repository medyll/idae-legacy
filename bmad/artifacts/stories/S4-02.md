# Story S4-02 — Integration: JsonData Regression

**Epic:** Testing
**Sprint:** 04
**Points:** 3
**Priority:** Must

---

## User story

As a tester, add an integration PHPUnit that asserts `services/json_data.php` returns the same top-level JSON shape as the legacy implementation for at least three tables (`produit`, `agent`, `appscheme`).

---

## Acceptance criteria

- `idae/web/tests/Integration/JsonDataRegressionTest.php` present and runnable
- Tests assert `success`, `total`, and `rows` (or `row`) keys and validate seeded data present
- Tests run against `mongo-test` sidecar and do not touch production
