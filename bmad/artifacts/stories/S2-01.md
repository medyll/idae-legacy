# Story S2-01 – Convert remaining 26 `.less` files to `.scss`

**Epic:** SCSS Migration Complete  
**Sprint:** 02  
**Points:** 5  
**Priority:** Must  
**Traces to:** SCSS migration handoff-2026-03-02.md

---

## User Story

As a **front-end developer**, I want all 26 remaining LESS source files converted to SCSS,  
so that the stylesheet pipeline is fully modernized and the legacy `lessc` compiler can be deprecated.

---

## Context

Sprint 01 converted 3 core LESS files (`_vars.scss`, `_app_fabric.scss`, `_style.scss`). This story covers the remaining 26 files across 6 groups:

- **Group A (12 files):** Pure CSS — copy + rename only (vendor + simple stylesheets)
- **Group B (2 files):** LESS variables usage — add `@use` forward + `@var` → `$var` replacements
- **Group C (4 files):** LESS imports + mixin calls — fix `@import` → `@use` directives
- **Group D (3 files):** Files with own local LESS variables — convert `@var` → `$var`
- **Group E (2 files):** LESS loops/mixins — convert recursive loops to SCSS `@for` and LESS mixins to `@mixin` definitions
- **Group F (3 files):** Entry points with dynamic `@{location_fragment}` — deferred to S2-02

See `bmad/artifacts/handoff-2026-03-02.md` for complete file list and conversion details per group.

## Execution Plan

### Phase 1: Group A — Copy 12 vendor/pure-CSS files (no conversion needed)
Files to copy from `idae/web/appcss/less/` to `idae/web/appcss/scss/`:
- `vendor/animate-nc.less` → `vendor/_animate-nc.scss`
- `vendor/swiper/core.less` → `vendor/swiper/_core.scss`
- `vendor/swiper/effects.less` → `vendor/swiper/_effects.scss`
- `vendor/swiper/navigation.less` → `vendor/swiper/_navigation.scss`
- `vendor/swiper/navigation-f7.less` → `vendor/swiper/_navigation-f7.scss`
- `vendor/swiper/preloader.less` → `vendor/swiper/_preloader.scss`
- `vendor/swiper/preloader-f7.less` → `vendor/swiper/_preloader-f7.scss`
- `vendor/swiper/scrollbar.less` → `vendor/swiper/_scrollbar.scss`
- `vendor/swiper/swiper.less` → `vendor/swiper/_swiper.scss`
- `vendor/swiper/zoom.less` → `vendor/swiper/_zoom.scss`
- `sortable.less` → `_sortable.scss`
- `swiper-override.less` → `_swiper-override.scss`

### Phase 2: Group B — Add `@use` and convert variables (2 files)
- `css/global-tokens.less` → `css/_global-tokens.scss`: uses `@font_inter`, `@font-size-base`, `@color-text`, `@btn-*`, `@input-*`, `@heading-weight`
- `pikaday.less` → `_pikaday.scss`: uses `@font_inter`

Add header and replace all `@var` with `$var`:
```scss
@use "vars" as *;
```

### Phase 3: Group C — Fix imports in 4 files (no mixin calls detected)
- `css/table.less` → `css/_table.scss`
- `css/flex_old.less` → `css/_flex-old.scss`
- `interface.less` → `_interface.scss`
- `notifier.less` → `_notifier.scss`

Convert `@import "vars/app_fabric"` → `@use "vars/app_fabric" as *;`

### Phase 4: Group D — Convert local variables in 3 files
- `skin/skin_default.less` → `skin/_skin-default.scss`: `@color_1..@color_12`, `@font-family_1` → `$color_1...$color_12`, `$font-family_1`
- `skin/skin_seven.less` → `skin/_skin-seven.scss`: `@color_1..@color_4` → `$color_1...$color_4`
- `app_window_gui/app_window_gui.less` → `app_window_gui/_app-window-gui.scss`: `@color0..@color1` → `$color0...$color1`

### Phase 5: Group E — Convert LESS loops and mixins (2 complex files)
- `css/flex.less` → `css/_flex.scss`:
  - Convert recursive loops `.generate-flex_shrink_grow(@n, @i)` to SCSS `@for` loops
  - Example: LESS `.generate-flex_shrink_grow(12, 1)` becomes `@for $i from 1 through 12 { ... }`

- `vendor/swiper/mixins.less` → `vendor/swiper/_mixins.scss`:
  - Convert LESS mixin definitions (`.preserve3d()`, `.encoded-svg-background()`) to `@mixin`/`@include`
  - Example: LESS `.preserve3d() { transform: preserve-3d; }` becomes `@mixin preserve3d { transform: preserve-3d; }`

### Phase 6: Build and Verify
- Run: `cd idae/web && npm run build:css`
- Check for compilation errors in all entrypoints
- Compare generated CSS output with prior LESS compilation (defer detailed diff to S2-02)
